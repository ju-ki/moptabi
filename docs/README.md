## プロジェクト概要

AI旅行計画プランナーは、Google Maps APIを活用して効率的な旅行プランを作成・管理できるWebアプリケーションです。

### 主要機能
- 旅行プランの作成・編集・管理
- Google Maps APIを使用した観光スポット検索
- 最適なルートとスケジュールの提案
- ドラッグ&ドロップによるスポット順序変更
- リアルタイムでの移動時間・コスト計算

### 技術スタック
- **Frontend**: Next.js, React, TypeScript, Tailwind CSS
- **Backend**: Hono, Prisma, PostgreSQL
- **認証**: Clerk
- **地図**: Google Maps API
- **状態管理**: Zustand

## 開発環境の起動

```bash
# リポジトリクローン
git clone <repository-url>
cd ai-travel-planner

# 環境変数設定
cp frontend/.env.example frontend/.env.local
cp backend/.env.example backend/.env

# Docker環境起動
docker-compose up -d

# データベース初期化
docker-compose exec backend npx prisma migrate dev
```
